# 国際化 (I18n)

すべてのテキストはローカライズファイルに保存されています。新しいロケールで変更・追加できます。

## 利用可能なロケール

| 言語 | ファイル |
|------|----------|
| ドイツ語 | [de.yml](https://github.com/servactory/servactory/blob/main/config/locales/de.yml) |
| 英語 | [en.yml](https://github.com/servactory/servactory/blob/main/config/locales/en.yml) |
| スペイン語 | [es.yml](https://github.com/servactory/servactory/blob/main/config/locales/es.yml) |
| フランス語 | [fr.yml](https://github.com/servactory/servactory/blob/main/config/locales/fr.yml) |
| イタリア語 | [it.yml](https://github.com/servactory/servactory/blob/main/config/locales/it.yml) |
| ロシア語 | [ru.yml](https://github.com/servactory/servactory/blob/main/config/locales/ru.yml) |
