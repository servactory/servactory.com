# リリース 2.9

以下の変更が準備され、実装されました。

## Datory

Datoryオブジェクトのネイティブサポートが追加されました。

## テスト

### RSpec

`allow_*`ヘルパーの`with`オプションのサポートが追加されました。[詳細](../guide/testing/rspec/legacy#contains)。

以前に実装された機能が改善されました。

## Ruby

Ruby 3.4 Preview 2のサポートが検証されました。

## Rails

Rails 8.0のサポートが追加されました。
Rails 8.0では、Rubyバージョン3.1はサポートされていません。
