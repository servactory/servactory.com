# リリース 2.6

以下の変更が準備され、実装されました。

## アトリビュート

### オプション

#### `consists_of`の変更

`consists_of`オプションが[ダイナミックオプション](../guide/options/dynamic#consists-ofオプション)になりました。

#### `service_class_name`の変更

一部のオプションで利用可能な`service_class_name`アトリビュートが、新しい`service`アトリビュートに置き換えられました。

新しいアトリビュートは、準備されたデータセットを含むオブジェクトです: `class_name`。

また、翻訳用のメソッドも含まれます: `translate`。

## コンフィグレーション

### `i18n_root_key`の追加

ルート翻訳キーの名前を変更するための`i18n_root_key`コンフィグレーションが追加されました。
デフォルト値は`servactory`です。

::: code-group

```ruby [app/services/application_service/base.rb]
configuration do
  i18n_root_key :datory
end
```

:::
