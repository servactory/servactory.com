# Release 2.13

The following changes have been prepared and implemented.

## Attributes

### Options

#### Improved `type`

Fixed the `type` option behavior with default values when using `TrueClass` and/or `FalseClass` in input.

#### Improved `schema`

Added support for the `prepare` option inside the `schema` option.

Fixed the `schema` option behavior with default values when used in input.

#### Improved `inclusion`

Fixed the `inclusion` option behavior with default values when used in input.
